# Contacts API
A simple contacts Nodejs API with mysql database connection.
Este es un ejemplo de texto.

## Development setup

```bash
git clone https://github.com/soycmramos/contacts-api.git
```

```bash
cd contacts-api
```

```bash
yarn install
```

Init your MySQL server (XAMPP/WAMP)

```bash
yarn dev
```

# Testing

```bash
yarn test
```
## ENV File

Check `.env.example` file

```properties
PORT
DB_HOST
DB_USER
DB_PASSWORD
DB_NAME
```
