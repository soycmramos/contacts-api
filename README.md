# Contacts API
A simple contacts Nodejs API with mysql database connection and JWT validation.

## Development setup

```bash
git clone https://github.com/soycarlosramos/contacts-api.git
```

```bash
cd contacts-api
```

```bash
yarn install
```

```bash
yarn dev
```

# Testing

```bash
yarn test
```
## ENV File

Check `.example` file

```properties
PORT
JWT_SECRET
DB_HOST
DB_USER
DB_PASSWORD
DB_NAME
```
